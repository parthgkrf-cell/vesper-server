Excel File Format

Ensure your file has these column headers on the first sheet:

  • Name — Recipient's name (e.g. John Doe)
  • Email — Recipient's address (e.g. john@example.com)
  • Affiliation — Company/Organization (e.g. Google)
  • Date — Custom date tag (e.g. 2026-07-15)
  • Technical Schedual Name — Task name (e.g. System Maintenance)
  • Time — Custom time tag (e.g. 10:00 AM)
  • CC — Carbon copy emails (comma-separated)
  • BCC — Blind carbon copy emails (comma-separated)
  • Sender — Relay email address (SMTP matching)
  • Action — send or draft
  • S_Time — Mailing Scheduled Time (e.g. 2026-07-02 15:30:00). Used to schedule dispatch but *never* sent inside email.

Getting Started in 3 Steps

  1. Import Recipients: Prepare an Excel or CSV file. Ensure it contains the headers Name, Email, Affiliation, and Time. Drag it into the Import Excel workspace.
  2. Compose Your Template: Go to Compose Template and draft your message. Insert merge tags like {Name} or {Affiliation} to personalize the content. Look at the Personalized Live Preview on the right to make sure it looks perfect.
  3. Configure & Start: Select your sending method in Configuration. Use the secure Desktop Mail Client (mailto:) method by default (opens your native mail app), or configure automated sending. Click Start Scheduler in the sidebar.

Understanding Sending Modes

Mailto: Mode (No Setup)

When the scheduled time arrives, Vesper triggers a system link that pops open your default email client (Outlook, Mail, Gmail Web) with the recipient's address, subject line, and personal body already filled in. You just review and click "Send". Perfect for direct, non-bulk personal emails.

API Modes (Brevo / Resend)

Sends background emails automatically via REST HTTP without opening any desktop client. Extremely fast and fully automated. Best for hands-free background queues. Setup takes 2 minutes on Brevo or Resend for free API keys.

Local Server Mode (SMTP)

If you want standard SMTP automation (e.g. logging into Gmail App Passwords directly or custom company relays), run our local python server. Vesper communicates with it securely to dispatch emails in the background.

Frequently Asked Questions

Do I need to keep the tab open?

Yes. Since the scheduler runs directly in your browser, you must keep this browser tab open to allow the background timer to process your active scheduled queue.

What happens if the scheduled time has already passed when I import?

The scheduler will treat it as "due immediately" and trigger/send it within the first few seconds of starting the scheduler engine.

Can I edit individual rows after importing?

You can manage the active status of scheduled items in the Schedule & Queue tab, check/uncheck items, trigger specific items immediately, or reset them.

Are my passwords and API keys secure?

Absolutely. All configuration, keys, and SMTP credentials are stored locally in your browser's localStorage. No data is sent to Vesper or any external servers except the email service providers you configure.
